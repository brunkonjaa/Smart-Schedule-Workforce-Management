# Smart Schedule

## Project Summary

Smart Schedule is a hospitality workforce management system focused on weekly rota planning for small teams. The MVP covers staff management, availability submission, leave requests, shift assignment, conflict checks, swap requests, reporting, and audit logging.

## Tech Stack

1. Frontend: HTML, CSS, JavaScript
2. Frontend support: Bootstrap, limited and customized
3. Backend: Node.js with Express
4. Database: MySQL
5. Authentication: session-based authentication with express-session and bcrypt
6. Database access: mysql2 with prepared statements
7. Testing: Jest, Supertest, Postman, manual UAT

## Folder Structure

1. `backend/` Node.js and Express application
2. `frontend/` HTML, CSS and JavaScript application
3. `database/` migrations, schema, and seed files
4. `docs/` planning, architecture, design, requirements, and testing documents
5. `assets/` screenshots and report assets
6. `scripts/` utility scripts
7. `infra/` infrastructure and CI support files
8. `research/` supporting research notes
9. `wireframes/` UI sketches and layout references
10. `logs/` local log output if needed
